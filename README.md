🎒 Campus Lost & Found

A modern full-stack web application built to help students easily report, browse, and recover lost or found items within a campus environment 🏫✨

Whether it’s a lost ID card, misplaced headphones, forgotten water bottle, or found gadgets — this platform creates a simple and secure way for students to reconnect with their belongings 🔍📦

---

🚀 Features

- ✨ User Registration & Secure Login
- 🔐 JWT Authentication & Password Encryption
- 📤 Post Lost or Found Items
- 🖼️ Upload Item Images
- ✏️ Edit & Delete Posts
- ✅ Claim Found Items
- 📞 Contact Details Sharing After Claiming
- 👤 User Profile Management
- 📊 Dashboard & My Posts Section
- 🎨 Modern Responsive UI
- ☁️ Cloud Image Storage with Cloudinary
